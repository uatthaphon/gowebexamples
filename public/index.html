เรียนรู้การสร้างเว็บไซต์ด้วยภาษา Go จากตัวอย่าง

Go เป็นภาษาโปรแกรมแบบโอเพนซอร์ซที่ถูกออกแบบมาเพื่อให้สามารถสร้างซอฟต์แวร์ได้อย่างง่ายดาย รวดเร็วและเชื่อถือได้ เราสามารถที่จะดู รายชื่อบริษัท ที่เป็นที่รู้จักกันในวงกว้าง พวกเขาใช้ภาษา Go ในการสร้างสินค้าและบริการเพื่อเพิ่มประสิทธิภาพให้กับบริษัทของพวกเขา

Go เว็บตัวอย่าง เป็นบทเรียนที่สามารถทำความเข้าใจได้ง่าย จากการดูโค้ดตัวอย่างที่มีให้ในบทเรียนสำหรับการสร้างเว็บไซต์ หากพร้อมแล้วพวกเราก็มาเริ่มเรียนรู้การสร้างเว็บไซต์ด้วยภาษา Go กันดีกว่า เว็บไซต์นี้ ได้รับแรงบันดาลใจมาจาก Go By Example ซึ่งเหมาะมากสำหรับการเริ่มต้นเรียนรู้พื้นฐานภาษา Go

ถ้าคุณกำลังเรียนรู้การสร้างเว็บไซต์ด้วยภาษา Go หรือกำลังเริ่มต้นที่จะเรียนรู้ คุณก็มาถูกที่แล้ว ที่นี่มีตัวอย่างโค้ดและบทเรียนที่ดี เป้าหมายในการสร้างบทเรียนนี้คือ การสร้างตัวอย่างที่เข้าใจง่ายและให้ข้อมูลที่ชัดเจน เพื่อให้คุณสามารถกลายเป็นนักพัฒนาเว็บไซต์ภาษา Go (Go web developer!) เนื้อหาของ Go เว็บตัวอย่าง ครอบคลุมความรู้พื้นฐานการสร้างเว็บไซต์ ตั้งแต่เรื่องของ routers และ templates ไปจนถึงเรื่อง middleware และ websockets คุณจะได้เรียนรู้จากโค้ดที่เป็นระบบและเนื้อหาที่ง่ายต่อการเรียนรู้ พร้อมด้วยรายละเอียดเพื่อความเข้าใจ ในแต่ละเนื้อหาของบทเรียน

เรามาเริ่มต้นกันเลยดีกว่า ขั้นแรกเข้าไปดู ตัวอย่างแรก ตัวอย่างนี้แสดงวิธีสร้างหน้าเว็บสุดคลาสสิกอย่าง "Hello World" หรือถ้าคุณรู้วิธีอยู่แล้วก็สามารถข้ามไปดูตัวอย่างการสร้าง Routing (ซึ่งใช้แพ็กเกจ gorilla/mux) แทนได้
